# SMS-BirthdayWisher
<p> This python program will wish your friends at 12 am on their birthday by sending a SMS. </p>

# Developed Using:
<ul>
  <li> python 3.8.4 </li>
  <li> openpyxl module </li>
  <li> twilio module </li>
  <li> time module </li>
  <li> datetime module </li>
</ul>
